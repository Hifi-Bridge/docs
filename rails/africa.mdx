## Individual

The Africa Rail for individual users requires KYC (Know Your Customer) verification. The requirements vary by country.

### KYC Fields

| Field           | Description                                                                                                                                                     | `Required` |
| :-------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------- |
| **firstName**   | User's first name                                                                                                                                               | `Required` |
| **lastName**    | User's last name                                                                                                                                                | `Required` |
| **email**       | User's email address                                                                                                                                            | `Required` |
| **phone**       | User's phone number                                                                                                                                             | `Required` |
| **address**     | The address where the user currently resides. Required fields: `addressLine1`, `city`, `stateProvinceRegion`, `postalCode`, `country`. Optional: `addressLine2` | `Required` |
| **dateOfBirth** | User's date of birth                                                                                                                                            | `Required` |
| **idType**      | Type of identification document. Refer to the [ID Types](#id-types) section                                                                                     | `Required` |
| **idNumber**    | Identification document number                                                                                                                                  | `Required` |

<Accordion title="ID Types">

| ID Type          | Description                  |
| :--------------- | :--------------------------- |
| DRIVERS          | Drivers license              |
| ID_CARD          | Identification card          |
| PASSPORT         | National passport            |
| RESIDENCE_PERMIT | Residence permit or document |

</Accordion>

### Nigeria Requirements

For Nigerian users, the following additional fields are required:

| Field                  | Description                             | `Required` |
| :--------------------- | :-------------------------------------- | :--------- |
| **additionalIdType**   | Additional identification document type | `Required` |
| **additionalIdNumber** | Additional identification number        | `Required` |

## Business

### KYB Fields

| Field                       | Description                         | `Required` |
| :-------------------------- | :---------------------------------- | :--------- |
| **businessName**            | The full legal name of the business | `Required` |
| **taxIdentificationNumber** | Company tax number                  | `Required` |

<Warning>
  **Note:** Business accounts are not currently supported in Nigeria—only
  individual accounts are available.
</Warning>
