## Inflows

Inflow products help you create entry points for value so funds can move into your system from bank rails and digital assets.

| Product | Description | Use this to |
| :-- | :-- | :-- |
| **Virtual Accounts** | Unique pass-through accounts in supported markets and currencies — receive fiat deposits and programmatically convert to stablecoin. | Accept recurring inbound bank transfers and reconcile funds with stablecoin. |
| **Onramps** | Convert fiat into supported stablecoin balances through programmatic transfer flows. | Bring user or treasury funds on-chain from local bank rails. |
| **Wallet Deposits** | Provision wallets that can receive and hold supported digital assets. | Let users or your business receive onchain assets. |

## Orchestration

Orchestration products give you programmable control over how money moves across accounts, wallets, networks, and workflows.

| Product | Description | Use this to |
| :-- | :-- | :-- |
| **Wallet Transfers** | Manage digital asset movement between wallets. | Power treasury, internal rebalancing, and operational crypto movements. |
| **Batch Transfers** | Submit grouped transfer in an atomic orchestration workflow. | Execute multiple transactions atomically and cost-efficiently. |
| **Bridging** | Move assets across supported chains. | Route liquidity or settle value across multiple networks. |
| **Swaps** | Exchange supported assets. | Convert one digital asset into another as part of a payment or treasury flow. |
| **Settlement Rules** | Settlement logic for virtual account deposits. | Automate value routing and reduce manual operations in your business logic. |
| **Orchestration Addresses** | A static onchain address that orchestrates fund flows with flexible configuration | Move and convert funds automatically based on predefined rules. |
| **Transfer Approvals** | Add policy-based approval steps before transfer execution. | Enforce internal controls and governance for sensitive fund movement. |

## Outflows

Outflow products help you settle to users, partners, and counterparties across bank rails and digital asset destinations.

| Product | Description | Use this to |
| :-- | :-- | :-- |
| **Offramps** | Convert stablecoin balances into fiat and deliver to banks via multiple payment rails. | Settle to beneficiaries in fiat currencies. |
| **Visa Direct (Push to Card)** | Settle fiat globally and instantly to cards via the Visa network. | Settle payments in fiat when speed matters |

## Private Transactions

For institution-grade settlement between trusted counterparties, HIFI also supports private transaction workflows designed for controlled, compliant asset movement.

| Product | Description | Use this to |
| :-- | :-- | :-- |
| **Canton** | Enterprise privacy enabled blockchain built for compliant, programmable financial workflows. | Connect to privacy-enabled institutional blockchain infrastructure. |

## Compliance

HIFI includes built-in onboarding and compliance primitives so product teams can launch regulated money movement with fewer integration layers.

| Product | Description | Use this to |
| :-- | :-- | :-- |
| **KYC/KYB** | Collect and evaluate identity and business verification requirements. | Satisfy onboarding controls before enabling production money movement. |
| **Compliance Links** | Generate hosted onboarding flows for compliance submission. | Offload document collection and verification UX while keeping API control. |
| **Token Sharing** | Share your end users' KYC information via a token provided by an eligible compliance vendor. | Enable faster end-user onboarding with less integration overhead. |
