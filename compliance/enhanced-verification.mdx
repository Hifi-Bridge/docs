Onboarding business users from certain countries requires Enhanced Verification. HIFI is currently running this flow as a pilot program with an initial set of customers.

<Warning>
  Enhanced Verification onboarding is part of a pilot program. Contact HIFI
  before enabling USD payouts for customers in these countries.
</Warning>

## Pilot Countries

| Country | ISO Code | Rail | Direction | Payment Method |
| :-- | :-- | :-- | :-- | :-- |
| India | IN | USD | Payout | SWIFT |
| Nigeria | NG | USD | Payout | SWIFT |
| Panama | PA | USD | Payout | SWIFT |
| Philippines | PH | USD | Payout | SWIFT |
| Saudi Arabia | SA | USD | Payout | SWIFT |
| Vietnam | VN | USD | Payout | SWIFT |

<Info>
  These countries are not prohibited regions. Business users from these
  countries require Enhanced Verification before USD payouts can be enabled.
</Info>

## What Changes

<CardGroup cols={2}>
  <Card title="Entity documents" icon="file-lines">
    Baseline business document collection and review do not change.
  </Card>

  <Card title="UBO threshold" icon="users">
    All UBOs with 10% ownership or above must complete full individual KYC
    before the entity can be approved.
  </Card>

  <Card title="Liveness verification" icon="camera">
    Qualifying UBOs must complete live ID capture and biometric liveness
    verification. Static ID uploads and screenshots are not accepted.
  </Card>

  <Card title="$10,000 review hold" icon="dollar-sign">
    Any single transaction or payment of $10,000 USD or more is held for review
    before processing.
  </Card>
</CardGroup>
