Convert Mexican Pesos (MXN) to USDC using HIFI's onramp functionality. This guide shows you how to set up and execute MXN to USDC conversions.

## Overview

MXN to USDC onramps are perfect for:

- **Mexican Users**: Converting pesos to stablecoins
- **Cross-border Payments**: Sending money from Mexico to international recipients
- **Crypto Adoption**: Easy entry into the crypto ecosystem
- **Remittances**: Sending money abroad from Mexico

## Prerequisites

Before starting, ensure you have:

- A HIFI user account with completed KYC
- Mexican bank account or payment method
- Valid identification documents

## Creating an Onramp

### Basic MXN to USDC Conversion

```bash
curl --request POST \
     --url https://sandbox.hifibridge.com/v2/onramps \
     --header 'Authorization: Bearer YOUR_API_KEY' \
     --header 'accept: application/json' \
     --header 'content-type: application/json' \
     --data '{
  "sourceCurrency": "mxn",
  "destinationCurrency": "usdc",
  "destinationChain": "POLYGON",
  "amount": "1000.00",
  "userId": "your-user-id"
}'
```

## Monitoring Onramp Status

### Check Status

```bash
curl --request GET \
     --url https://sandbox.hifibridge.com/v2/onramps/{onrampId} \
     --header 'Authorization: Bearer YOUR_API_KEY'
```

### Status Updates

Onramps progress through these statuses:

- **PENDING**: Waiting for payment
- **PAYMENT_RECEIVED**: Payment confirmed, processing conversion
- **PROCESSING**: Converting MXN to USDC
- **COMPLETED**: USDC delivered to wallet
- **FAILED**: Conversion failed (check reason)

## Webhook Events

Listen for onramp events:

```javascript
// Payment received
webhook.on("ONRAMP.PAYMENT.RECEIVED", (data) => {
  console.log("Payment received for onramp:", data.onrampId);
});

// Conversion completed
webhook.on("ONRAMP.COMPLETED", (data) => {
  console.log("MXN to USDC conversion completed:", data.onrampId);
  console.log("USDC amount:", data.convertedAmount);
});
```

## Best Practices

### Amount Validation

```javascript
// Validate minimum amounts
const minimumAmount = 100; // MXN
if (amount < minimumAmount) {
  throw new Error("Minimum amount is 100 MXN");
}

// Check daily limits
const dailyLimit = 10000; // MXN
const userDailyTotal = await getUserDailyOnramps(userId);
if (userDailyTotal + amount > dailyLimit) {
  throw new Error("Daily limit exceeded");
}
```

### Error Handling

```javascript
try {
  const onramp = await createOnramp({
    sourceCurrency: "mxn",
    destinationCurrency: "usdc",
    amount: "1000.00",
    userId: userId,
  });

  if (onramp.status === "PENDING") {
    // Display payment instructions
    displayPaymentInstructions(onramp.paymentInstructions);
  }
} catch (error) {
  console.error("Onramp creation failed:", error);

  if (error.code === "INSUFFICIENT_KYC") {
    // Handle KYC requirements
  } else if (error.code === "LIMIT_EXCEEDED") {
    // Handle limit exceeded
  }
}
```

## Fees and Limits

### Fee Structure

- **Conversion Fee**: 0.5% of transaction amount
- **Network Fee**: Variable based on blockchain congestion
- **Processing Fee**: 15 MXN per transaction

### Limits

- **Minimum Amount**: 100 MXN
- **Maximum Amount**: 50,000 MXN per transaction
- **Daily Limit**: 100,000 MXN per user
- **Monthly Limit**: 1,000,000 MXN per user

## Compliance

### KYC Requirements

Mexican users must provide:

- Valid Mexican ID (INE)
- Proof of address (utility bill)
- Bank account verification
- Tax identification (RFC)

### AML Monitoring

All transactions are monitored for:

- Unusual patterns
- High-value transfers
- Multiple small transactions
- Cross-border activity
