# Send USD to MXN

Learn how to convert between fiat currencies using HIFI's onramp and offramp services.

## Overview

HIFI's onramp and offramp services allow you to convert between fiat currencies and stablecoins using local payment methods and bank rails. This enables:

- Cross-border payments
- Fiat to crypto conversions
- Local payment method support
- Competitive exchange rates

## Prerequisites

- API credentials configured
- Source wallet with USD balance
- Destination wallet for MXN
- KYC verification completed

## Basic fiat conversion

```javascript
const conversion = await hifi.onramps.create({
  from_currency: "USD",
  to_currency: "MXN",
  amount: "1000.00",
  from_wallet: "wallet_123",
  to_wallet: "wallet_456",
  payment_method: "bank_transfer",
});
```

## Advanced conversion with routing

```javascript
const conversion = await hifi.onramps.create({
```

## Get conversion quote

```javascript
// Get a quote before converting
const quote = await hifi.onramps.getQuote({
  from_currency: "USD",
  to_currency: "MXN",
  amount: "1000.00",
  payment_method: "bank_transfer",
});

console.log(`Exchange rate: 1 USD = ${quote.rate} MXN`);
console.log(`Expected output: ${quote.expected_output} MXN`);
console.log(`Processing time: ${quote.estimated_time} hours`);
console.log(`Fees: ${quote.fee} USD`);

// Execute the conversion
const conversion = await hifi.onramps.create({
  from_currency: "USD",
  to_currency: "MXN",
  amount: "1000.00",
  from_wallet: "wallet_123",
  to_wallet: "wallet_456",
  payment_method: "bank_transfer",
  quote_id: quote.id,
});
```

## Supported currencies

### Fiat currencies

- USD, EUR, GBP, CAD, AUD
- MXN, BRL, ARS, CLP, COP
- JPY, KRW, SGD, HKD, INR

### Payment methods

- Bank transfers (ACH, SEPA, SWIFT)
- Credit/debit cards
- Digital wallets (PayPal, Apple Pay, Google Pay)
- Local payment methods (OXXO, Boleto, etc.)

## Processing times

Typical processing times by payment method:

- **Bank transfers**: 1-3 business days
- **Credit/debit cards**: Instant to 1 hour
- **Digital wallets**: 5-30 minutes
- **Local methods**: 1-24 hours

## Fees and limits

### Fees

- **Bank transfers**: 0.5-1.5% of amount
- **Cards**: 2.5-3.5% of amount
- **Digital wallets**: 1.0-2.0% of amount
- **Local methods**: 1.5-2.5% of amount

### Limits

- **Minimum**: $10 USD equivalent
- **Maximum**: $50,000 USD equivalent per transaction
- **Daily limit**: $100,000 USD equivalent per user
- **Monthly limit**: $500,000 USD equivalent per user

## KYC requirements

Different limits based on verification level:

- **Basic**: $1,000 daily limit
- **Enhanced**: $10,000 daily limit
- **Premium**: $50,000 daily limit

## Error handling

Common conversion errors:

- **Insufficient balance**: Check wallet has enough funds
- **KYC required**: Complete identity verification
- **Unsupported currency**: Check supported currency list
- **Payment method declined**: Try alternative payment method

## Monitoring progress

```javascript
// Check conversion status
const status = await hifi.onramps.get("conversion_abc123");

// Get transaction details
console.log(`Status: ${status.status}`);
console.log(`Expected completion: ${status.estimated_completion}`);
```

## Best practices

1. **Complete KYC early** to avoid delays
2. **Check exchange rates** before large conversions
3. **Use appropriate payment methods** for your needs
4. **Keep transaction IDs** for support requests
