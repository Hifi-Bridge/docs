# Bridge ETH to SOL

Learn how to move stablecoins from Ethereum to Solana using HIFI's cross-chain bridging.

## Overview

Chain bridging allows you to move assets across different blockchain networks while keeping your balances unified in your HIFI wallet. This enables you to:

- Access different DeFi ecosystems
- Take advantage of lower fees on other chains
- Diversify across multiple networks
- Maintain unified balance management

## Prerequisites

- API credentials configured
- Source wallet with USDC on Ethereum
- Destination wallet on Solana (can be the same wallet)

## Basic bridge

```javascript
const bridge = await hifi.bridges.create({
  from_network: "ethereum",
  to_network: "solana",
  currency: "USDC",
  amount: "1000.00",
  wallet: "wallet_123",
});
```

## Advanced bridge with custom routing

```javascript
const bridge = await hifi.bridges.create({
  from_network: "ethereum",
  to_network: "solana",
  currency: "USDC",
  amount: "1000.00",
  wallet: "wallet_123",
  routing_preference: "fastest", // or "cheapest"
  slippage_tolerance: 1.0, // 1% max slippage
  memo: "Moving to Solana for DeFi",
});
```

## Get bridge quote

```javascript
// Get a quote before bridging
const quote = await hifi.bridges.getQuote({
  from_network: "ethereum",
  to_network: "solana",
  currency: "USDC",
  amount: "1000.00",
});

console.log(`Estimated time: ${quote.estimated_time} minutes`);
console.log(`Bridge fee: ${quote.fee} USDC`);
console.log(`Expected output: ${quote.expected_output} USDC`);

// Execute the bridge
const bridge = await hifi.bridges.create({
  from_network: "ethereum",
  to_network: "solana",
  currency: "USDC",
  amount: "1000.00",
  wallet: "wallet_123",
  quote_id: quote.id,
});
```

## Supported networks

HIFI supports bridging between:

- **Ethereum** ↔ Polygon, Arbitrum, Optimism, Base, Solana
- **Polygon** ↔ Ethereum, Arbitrum, Optimism, Base
- **Arbitrum** ↔ Ethereum, Polygon, Optimism, Base
- **Optimism** ↔ Ethereum, Polygon, Arbitrum, Base
- **Base** ↔ Ethereum, Polygon, Arbitrum, Optimism
- **Solana** ↔ Ethereum

## Supported currencies

- USDC
- USDT
- DAI
- BUSD

## Bridge times

Typical bridge completion times:

- **Ethereum ↔ L2s**: 5-15 minutes
- **L2 ↔ L2**: 2-10 minutes
- **Ethereum ↔ Solana**: 10-30 minutes
- **Cross-chain**: 15-45 minutes

## Fees

Bridge fees vary by route and are typically:

- **Ethereum ↔ L2s**: $5-15
- **L2 ↔ L2**: $2-8
- **Ethereum ↔ Solana**: $10-25
- **Cross-chain**: $15-40

## Monitoring progress

```javascript
// Check bridge status
const status = await hifi.bridges.get("bridge_xyz789");

// Get transaction hashes
console.log(`Source TX: ${status.source_transaction}`);
console.log(`Destination TX: ${status.destination_transaction}`);
```

## Error handling

Common bridge errors:

- **Insufficient balance**: Ensure wallet has enough funds
- **Network congestion**: Try during lower traffic periods
- **Invalid destination**: Verify destination network is supported
- **Bridge timeout**: Contact support if bridge takes >2 hours

## Best practices

1. **Check network status** before bridging
2. **Use appropriate slippage** for volatile conditions
3. **Monitor gas prices** for optimal timing
4. **Keep transaction hashes** for tracking
