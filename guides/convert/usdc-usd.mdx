# Offramps

Learn how to convert stablecoins to fiat currencies using HIFI's offramp services.

## Overview

HIFI's offramp services allow you to convert stablecoins to fiat currencies and withdraw to local bank accounts or payment methods. This enables:

- Crypto to fiat conversions
- Cross-border withdrawals
- Local bank account support
- Competitive exchange rates

## Supported currencies

### Fiat currencies

- **Americas**: USD, CAD, MXN, BRL, ARS, CLP, COP
- **Europe**: EUR, GBP, CHF, SEK, NOK, DKK
- **Asia-Pacific**: JPY, KRW, SGD, HKD, INR, AUD, NZD
- **Middle East & Africa**: AED, SAR, ZAR, EGP

### Stablecoins

- USDC (Ethereum, Polygon, Arbitrum, Optimism, Base, Solana)
- USDT (Ethereum, Polygon, Arbitrum, Optimism, Base, Solana)
- DAI (Ethereum, Polygon, Arbitrum, Optimism, Base)
- BUSD (Ethereum, BSC)

## Withdrawal methods

### Bank transfers

- **ACH** (US) - 1-3 business days
- **SEPA** (EU) - 1-2 business days
- **SWIFT** (International) - 2-5 business days
- **Local bank transfers** - 1-3 business days

### Digital wallets

- **PayPal** - 5-30 minutes
- **Apple Pay** - Instant to 15 minutes
- **Google Pay** - Instant to 15 minutes
- **Local wallets** - 5-60 minutes

### Local payment methods

- **OXXO** (Mexico) - 1-24 hours
- **Boleto** (Brazil) - 1-24 hours
- **Konbini** (Japan) - 1-24 hours
- **GCash** (Philippines) - 5-30 minutes

## Basic offramp

```javascript
const offramp = await hifi.offramps.create({
  from_currency: "USDC",
  to_currency: "USD",
  amount: "1000.00",
  wallet: "wallet_123",
  withdrawal_method: "bank_transfer",
  bank_account: "bank_456",
});
```

## Advanced offramp

```javascript
const offramp = await hifi.offramps.create({
```

## Get quote

```javascript
// Get a quote before converting
const quote = await hifi.offramps.getQuote({
  from_currency: "USDC",
  to_currency: "USD",
  amount: "1000.00",
  withdrawal_method: "bank_transfer",
});

console.log(`Exchange rate: 1 USDC = ${quote.rate} USD`);
console.log(`Expected output: ${quote.expected_output} USD`);
console.log(`Processing time: ${quote.estimated_time} hours`);
console.log(`Fees: ${quote.fee} USDC`);

// Execute the offramp
const offramp = await hifi.offramps.create({
  from_currency: "USDC",
  to_currency: "USD",
  amount: "1000.00",
  wallet: "wallet_123",
  withdrawal_method: "bank_transfer",
  bank_account: "bank_456",
  quote_id: quote.id,
});
```

## Fees and limits

### Fees by withdrawal method

- **Bank transfers**: 0.5-1.5% of amount
- **Digital wallets**: 1.0-2.0% of amount
- **Local methods**: 1.5-2.5% of amount

### Transaction limits

- **Minimum**: $10 USD equivalent
- **Maximum**: $50,000 USD equivalent per transaction
- **Daily limit**: $100,000 USD equivalent per user
- **Monthly limit**: $500,000 USD equivalent per user

### KYC verification levels

- **Basic**: $1,000 daily limit
- **Enhanced**: $10,000 daily limit
- **Premium**: $50,000 daily limit

## Bank account setup

Before using bank transfers, you need to add a bank account:

```javascript
const bankAccount = await hifi.bankAccounts.create({
  account_holder_name: "John Doe",
  account_number: "1234567890",
  routing_number: "021000021", // US ACH routing number
  bank_name: "Chase Bank",
  account_type: "checking", // or "savings"
  country: "US",
});
```

## Error handling

Common offramp errors:

- **Insufficient balance**: Check wallet has enough stablecoins
- **KYC required**: Complete identity verification
- **Unsupported currency**: Check supported currency list
- **Invalid bank account**: Verify bank account details
- **Daily limit exceeded**: Wait for next day or upgrade KYC

## Monitoring progress

```javascript
// Check offramp status
const status = await hifi.offramps.get("offramp_xyz789");

// Get transaction details
console.log(`Status: ${status.status}`);
console.log(`Expected completion: ${status.estimated_completion}`);
```

## Best practices

1. **Complete KYC early** to avoid delays
2. **Verify bank account details** before large withdrawals
3. **Check exchange rates** before converting
4. **Keep transaction IDs** for support requests
5. **Monitor processing times** for optimal timing
