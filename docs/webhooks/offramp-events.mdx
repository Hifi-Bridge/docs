---
title: "Offramp Events"
---

You can expect to receive webhook events whenever an offramp transfer is created or its status is updated.

## Types

- **OFFRAMP.CREATE**: Creation for offramps.
- **OFFRAMP.UPDATE**: Updates for offramps.

## Sandbox

<Note>
  In the Sandbox environment, we simulate the transaction status update process
  by sending a series of webhook events.
</Note>
