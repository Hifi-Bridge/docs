HIFI provides multiple ways to move money across different currencies, networks, and use cases. The core transfer types are **wallet transfers**, **onramps**, **offramps**, **swaps**, and **bridges**.

## Wallet Transfers

The simplest use case for sending money on HIFI is transferring funds between wallets on the same blockchain network.
