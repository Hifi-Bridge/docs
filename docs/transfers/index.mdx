HIFI provides multiple ways to move money across different currencies, networks, and use cases. The core transfer types are **wallet transfers**, **onramps**, and **offramps**.

## Wallet Transfers

The easiest way to send money on HIFI is by transferring funds between wallets on the same blockchain network.
